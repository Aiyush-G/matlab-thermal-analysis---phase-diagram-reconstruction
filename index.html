Thermal Analysis of Cooling Curves

Locates liquidus and solidus arrests in thermocouple cooling curves by background subtraction and derivative analysis, and builds experimental phase diagrams.

Download
Windows / macOS / Linux

Installation

  1. Download the .m file linked above.
  2. Run it. The MATLAB Runtime R2024b has been tested but previous versions should work.
  3. Launch the app and load the bundled sample dataset (sample_data/) to try it immediately.
